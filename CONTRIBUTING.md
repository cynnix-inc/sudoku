# Contributing

Follow project conventions in `.cursor/rules` and docs under `docs/`.

## Sub-Issue Templates
Use the sub-issue templates when breaking work down:
- docs/templates/feature-sub-issue.md
- docs/templates/bug-sub-issue.md
- docs/templates/chore-sub-issue.md

Include `Parent: #<epic-number>` in each sub-issue and link it as a sub-issue of the Epic.

## Development
- Use Node via Volta if configured.
- Run tests and lint locally before opening a PR.

## Code of Conduct
Be respectful and inclusive.
