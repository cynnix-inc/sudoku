How to play

Goal: Fill the grid so each row, column, and 3×3 box contains 1–9 with no repeats.

❤️ Lives

Lives are the number of wrong entries allowed. Each unique incorrect number placed in a cell uses one life. When you run out of lives, the game ends.

You can change Lives in Settings. Set to 11 for Unlimited. Daily puzzles use fixed Lives based on difficulty, so your setting won’t override Daily.

💡 Hints

Hints fill one correct cell. They’re limited by difficulty: Easy 5, Medium 3, Hard 2, Expert 1, Master/Extreme 0.

Each hint adds +30 seconds to your timer. Using any hint marks the run as assisted, so it won’t update your personal best time.

🧘 Zen mode

  • FocusHides the on‑screen timer, Lives, difficulty chip, and status messages.
  • LivesNormal games use Unlimited Lives in Zen. Daily puzzles keep their fixed Lives.
  • StatsZen runs don’t update wins/losses or best times.
  • ToggleTurn on/off in Settings. It takes effect immediately without interrupting your game.
