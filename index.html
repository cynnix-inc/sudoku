How to play

Goal: Fill the grid so each row, column, and 3×3 box contains 1–9 with no repeats.

📱 Mobile / Touch

  • 1–9Enter number (on‑screen pad)
  • 🧹 EraseClear cell
  • 👆 Double‑tapErase cell
  • ✎ (tap)Toggle Notes
  • ✎ (press‑and‑hold)Temporary Notes while held
  • Pad 1–9 (tap)Lock/unlock number; drag to paint
  • 📌 Long‑press cellAdd note for locked number
  • ⏱Pause/Resume (tap timer)
  • 💡Hint: fill one correct cell

⌨️ Mouse / Keyboard

  • 1…9Enter number
  • 0 / ⌫ / ⌦Erase (0, Backspace, Delete)
  • ← ↑ → ↓ / WASD / H J K LMove selection
  • Auto‑advanceAfter typing, moves forward; hold Shift to move backward
  • ✎ (click)Toggle Notes
  • ✎ (press‑and‑hold)Temporary Notes while held
  • Pad 1–9 (click)Lock/unlock number; drag to paint; ESC clears lock
  • Ctrl+ZUndo
  • Ctrl+Y / Ctrl+Shift+ZRedo
  • ⏱Pause/Resume (click timer)
  • 💡Hint: fill one correct cell
