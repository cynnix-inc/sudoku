# Ultimate Sudoku

## Quick start

- Install dependencies (Node 20.x via Volta is pinned):
  - `npm ci`
- Run the app:
  - Web: `npm run web`
  - Android: `npm run android`
  - iOS (on macOS): `npm run ios`
