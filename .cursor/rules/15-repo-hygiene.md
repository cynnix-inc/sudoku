# Repo Hygiene Rules

Applies to: entire repo

- Normalize line endings to LF via `.gitattributes`.
- Enforce consistent indentation and whitespace via `.editorconfig`.
- Avoid committing large binaries; prefer Git LFS (if needed) or external storage.
- Keep config files minimal and cross-OS friendly; avoid OS-specific paths.

References

- `.gitattributes` at repo root
- `.editorconfig` at repo root
